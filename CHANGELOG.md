# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2024-01-XX

### Added
- 🎉 Initial release
- ✨ Support for 5 leave scenarios:
  - Sick Leave (病假)
  - Personal Leave (事假)
  - Emergency (紧急情况)
  - Extended Leave (长期请假)
  - Academic Event (学术活动)
- 🌍 Bilingual support (English/Chinese)
- 📝 Professional email templates for each scenario
- 🤖 Interactive email generation workflow
- 💡 Writing tips and best practices
- 📚 Comprehensive examples documentation
- 🔧 Easy installation scripts for macOS/Linux and Windows

### Documentation
- README.md with quick start guide
- Examples.md with detailed usage examples
- Inline comments in SKILL.md for better understanding
